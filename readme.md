## Main Project
* HYUabot
  * Campus information system for Hanyang Univ. (2017~)
  * Provide a lot of information about campus
    * Shuttle Bus Timetable
    * Bus Timetable
    * Subway arrival information (Hanyang Univ at Ansan)
    * Cafeteria menu
    * Reading room seats
    * Academic calendar
    * Others

    * API Backend
